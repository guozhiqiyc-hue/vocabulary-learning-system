# 编辑器功能实现说明

## 实施状态 ✅

**完成时间**: 2026-02-24

## 功能概述

语块编辑器允许用户添加、编辑、删除和搜索语块，支持多条件筛选。

## 核心功能

### 1. 语块列表展示 ✅
- 显示所有语块的卡片列表
- 每个卡片显示：语块、释义、例句、标签
- 支持点击选中语块

### 2. 搜索和筛选 ✅
- **搜索框**: 按语块、释义、关键词搜索
- **等级筛选**: 全部/核心词汇/进阶词汇
- **主题筛选**: 按主题分类筛选

### 3. 添加语块 ✅
- 点击"添加语块"按钮打开表单
- 表单字段：
  - 语块 (必填)
  - 释义 (必填)
  - 例句 (必填) - 用 `**` 包裹关键词
  - 关键词 (必填) - 用逗号分隔
  - 等级 (核心词汇/进阶词汇)
  - 频率 (高频/中频/低频)
  - 主题 (支持输入建议)

### 4. 编辑语块 ✅
- 点击语块卡片上的 ✏️ 按钮
- 表单自动填充现有数据
- 保存后更新数据库

### 5. 删除语块 ✅
- 点击语块卡片上的 🗑️ 按钮
- 确认后删除语块

### 6. 批量编辑 (部分实现) 🔄
- 批量编辑按钮已添加
- 需要后续完善：
  - JSON 格式编辑
  - 预览和验证
  - 批量导入/导出

## 使用说明

### 添加新语块

1. 进入"编辑器"标签页
2. 点击"➕ 添加语块"按钮
3. 填写表单：
   ```
   语块: be good at doing
   释义: 擅长做某事
   例句: He is **good at** **playing** basketball.
   关键词: good, play
   等级: 核心词汇
   频率: 高频
   主题: 能力描述
   ```
4. 点击"添加"按钮

### 编辑现有语块

1. 在语块列表中找到要编辑的语块
2. 点击 ✏️ 按钮
3. 修改内容
4. 点击"保存"按钮

### 搜索语块

1. 在搜索框中输入关键词
2. 系统自动搜索并显示结果

### 筛选语块

1. 使用等级下拉框选择: 核心词汇/进阶词汇
2. 使用主题下拉框选择特定主题
3. 可以组合使用多个筛选条件

## 数据格式

### 例句格式

例句中的关键词需要用 `**` 包裹：

```
正确: He is **good at** **playing** basketball.
错误: He is good at playing basketball.
```

### 关键词格式

关键词用英文逗号分隔：

```
正确: good, play, basketball
错误: good play basketball
```

## 快捷操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + F` | 聚焦搜索框 |
| `ESC` | 关闭编辑表单 |

## 验证规则

- 语块: 必填，不能为空
- 释义: 必填，不能为空
- 例句: 必填，不能为空
- 关键词: 必填，至少一个

## 已知限制

1. **批量编辑**: 需要实现 JSON 格式编辑
2. **导入导出**: 需要实现批量导入/导出功能
3. **撤销操作**: 删除后无法撤销
4. **批量操作**: 不支持批量删除
