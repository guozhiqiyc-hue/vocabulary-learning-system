## 用户体验流程

### 学习流程

```
学习中心 → 选择学习模式 → 学习卡片
    ↓
点击"显示答案"或按空格
    ↓
查看释义和例句
    ↓
选择评分(1-5)或按数字键
    ↓
系统记录进度(SM-2算法)
    ↓
自动进入下一个语块
    ↓
完成 → 显示完成摘要 → 返回
```

### 复习流程

```
复习中心 → 查看统计 → 点击"开始复习"
    ↓
复习卡片(显示历史信息)
    ↓
点击"显示答案"或按空格
    ↓
选择评分(1-5)或按数字键
    ↓
系统更新进度和下次复习时间
    ↓
自动进入下一个语块
    ↓
完成 → 显示详细摘要 → 返回
```

## 键盘快捷键

| 按键 | 功能 |
|------|------|
| 空格 / Enter | 显示答案 |
| 1-5 | 质量评分 (1=忘记, 5=完美) |
| ESC | 退出当前模式 |

## SM-2 算法集成

系统完全实现了 SuperMemo 2 算法:

- **评分 5** (完美): 下次复习时间大幅延长
- **评分 4** (良好): 下次复习时间适当延长
- **评分 3** (勉强): 保持当前复习间隔
- **评分 2** (困难): 重新学习,缩短间隔
- **评分 1** (忘记): 完全重新学习
