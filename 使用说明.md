### 2. 模块化版本 (开发用)

```
D:\XX\WL\学习系统\02_English\词汇\每日词汇\vocabulary-learning-system-v6\index.html
```

- 代码未打包，方便调试

**使用方法:**
```bash

```

或手动启动服务器:
```bash
npx http-server -p 8080 -o
```

## 新功能说明

### 学习模式 📚

**入口:**
1. 今日学习计划 → "开始今日学习"
2. 学习标签页 → 选择学习模式

**三种学习模式:**
- **学习新语块** - 学习尚未接触的语块
- **继续学习** - 继续正在学习的语块
- **随机学习** - 随机抽取语块学习

**键盘快捷键:**
- `空格` / `回车` - 显示答案
- `1-5` - 快速评分
- `ESC` - 退出学习

---

### 复习模式 🔄

**入口:**
1. 今日学习计划 → "立即复习"
2. 复习标签页 → 查看待复习列表

**复习功能:**
- 到期语块自动提醒
- 复习历史信息展示
- 完成后详细摘要统计
- 质量评分分布图

**键盘快捷键:**
- `空格` / `回车` - 显示答案
- `1-5` - 快速评分
- `ESC` - 退出复习

---

## 重新打包代码

如果修改了源代码，需要重新打包 bundle.js:

```bash
npm run build
```

## 评分标准 (SM-2 算法)

| 评分 | 含义 | 效果 |
|------|------|------|
| ⭐⭐⭐⭐⭐ (5) | 完美记忆 | 下次复习时间大幅延长 |
| ⭐⭐⭐⭐ (4) | 良好记忆 | 下次复习时间适当延长 |
| ⭐⭐⭐ (3) | 勉强回忆 | 保持当前复习间隔 |
| ⭐⭐ (2) | 记忆困难 | 重新学习，缩短间隔 |
| ⭐ (1) | 完全忘记 | 完全重新学习 |

## 常见问题

**Q: 数据存储在哪里?**
A: 数据存储在浏览器的 IndexedDB 中，完全离线使用。

**Q: 如何备份数据?**
A: 点击顶部工具栏的 "💾 备份" 按钮下载 JSON 备份文件。

**Q: 如何重置学习进度?**
A: 目前需要在浏览器开发者工具中手动清除 IndexedDB 数据。
